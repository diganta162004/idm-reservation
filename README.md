# pg-tax-calculator

## Running the project

### Start with mock data
```shell
npm run start
```

Open URL
http://localhost:8080/

### Start with local dev
Local dev server port can be set in webpack.config.js inside devServer.proxy

default port: 5001
```shell
npm run start:dev
```

### Run test
```shell
npm run test
```

### Run test with watch and verbose
```shell
npm run test:watch
```

## Building the project
```shell
npm run build
```
